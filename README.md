# Rappi AI Engineer Technical Test

Este repositorio contiene la solución completa de la prueba técnica para el rol de AI Engineer.

## Proyectos incluidos

### 1. rappi-ops-ai
Sistema conversacional para consulta de métricas operativas y generación automática de insights.

### 2. competitive_intelligence
Pipeline de scraping y análisis competitivo sobre plataformas de delivery.

## Contenido
Cada proyecto incluye:
- código fuente
- instrucciones de ejecución
- outputs generados
- documentación

## Nota
En el caso de Competitive Intelligence se utilizó un enfoque híbrido humano-asistido para manejar restricciones reales como CAPTCHA.
